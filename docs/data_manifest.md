# Data manifest

## Cached embeddings

The main experiments use cached NumPy arrays under
`notebooks/_corpus_cache/`.

- `E_docs_e5small_1000000.npy`: 1,000,000 document embeddings,
  `intfloat/multilingual-e5-small`, shape `(1000000, 384)`.
- `E_docs_e5base_1000000.npy`: 1,000,000 document embeddings,
  `intfloat/multilingual-e5-base`, shape `(1000000, 768)`.
- `E_docs_mpnet_1000000.npy`: 1,000,000 document embeddings,
  `sentence-transformers/paraphrase-multilingual-mpnet-base-v2`,
  shape `(1000000, 768)`.
- `E_docs_e5large_1000000.npy`: 1,000,000 document embeddings,
  `intfloat/multilingual-e5-large`, shape `(1000000, 1024)`.
- `E_docs_bgem3_1000000.npy`: 1,000,000 document embeddings,
  `BAAI/bge-m3`, shape `(1000000, 1024)`.

The query files named `E_queries_*_self_q500.npy` contain 500 deterministic
self-retrieval query embeddings. Query relevance IDs are sampled with the
fixed seeds recorded in the experiment scripts.

## SVD and index artifacts

`notebooks/_corpus_cache/index_cache/` contains SVD bases, rotated arrays,
HNSW indexes and PQ indexes generated by the experiment scripts. These are
derived artifacts and can be rebuilt from the cached embeddings.

## Revision experiment

`notebooks/exp7_alignment_pq_leakage.py` uses:

- `E_docs_e5small_1000000.npy`
- `index_cache/svd_e5small_proj336.npz`
- deterministic sample seed `20260530`
- rotation seeds `11,23,31,47,53`

The exact sampled document IDs are stored in
`notebooks/exp7_outputs/exp7_doc_ids.npy`.

`notebooks/exp8_tradeoff_noise_sweep.py` uses:

- `E_docs_e5small_1000000.npy`
- `E_queries_e5small_self_q500.npy`
- `E_docs_e5base_1000000.npy`
- `E_queries_e5base_self_q500.npy`
- deterministic query seed `42`
- deterministic noise seed `2026`

The script writes the matched-distortion SVD/noise diagnostic to
`notebooks/exp8_outputs/exp8_tradeoff_summary.csv` and
`notebooks/exp8_outputs/exp8_tradeoff_summary.json`.

`notebooks/exp9_reference_corpus_attack.py` uses:

- `E_docs_e5small_1000000.npy`
- `index_cache/svd_e5small_proj336.npz`
- `corpus_wiki_ru_1000000.pkl`
- deterministic sample seed `20260531`
- rotation seeds `11,23,31,47,53`

The script writes the reference-corpus lookup stress test to
`notebooks/exp9_outputs/exp9_reference_attack_by_seed.csv`,
`notebooks/exp9_outputs/exp9_reference_attack_summary.csv`, and
`notebooks/exp9_outputs/exp9_reference_attack_summary.json`.

`notebooks/exp10_denoiser_significance.py` uses, for each encoder:

- `E_docs_<enc>_1000000.npy`
- `E_queries_<enc>_self_q500.npy`
- deterministic query/ground-truth seed `42` (numpy `default_rng`)
- bootstrap seed `2026`

It reproduces the Table 8 raw/SVD Acc numbers and writes the paired
significance results to `notebooks/exp10_outputs/exp10_significance.json`
and per-encoder hit vectors `notebooks/exp10_outputs/hits_<enc>.npz`.

`notebooks/exp11_beir_denoiser.py` uses the public Hugging Face datasets
`BeIR/scifact`, `BeIR/scifact-qrels`, `BeIR/nfcorpus`, `BeIR/nfcorpus-qrels`
and the e5 encoders from the Hub. Deterministic bootstrap seed `2026`;
encoding config `E11_MAXLEN=128`, `E11_BATCH=64`. It writes
`notebooks/exp11_outputs/exp11_beir.json`.
