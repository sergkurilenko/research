arXiv replacement source bundle

Title: Hybrid privacy-aware semantic search: SVD-truncated document geometry
and CKKS-encrypted query reranking under a restricted threat model

Author: Sergey Kurilenko

The source archive is deliberately flat. Compile with:

  pdflatex main.tex
  bibtex main
  pdflatex main.tex
  pdflatex main.tex

The archive includes the Springer Nature class/style files used for the
submitted rendering, the bibliography database and generated main.bbl, and all
six PDF figures. arXiv form fields are supplied separately in
arxiv_metadata.md, arxiv_abstract.txt, and arxiv_replacement_note.txt.
